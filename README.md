## 最初にやること

### node_modulesを取り込む
```bash
npm install
```

### サーバ起動
```bash
npm run dev
# or
yarn dev
```
[http://localhost:3000](http://localhost:3000)で開くはずです！

## ディレクトリ構成
srcの中で作業します。
ページに関するものは、`src/pages`の中に作成
コンポーネントは、`src/components`の中に作成
